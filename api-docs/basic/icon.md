# Icon 图标

## 介绍

基于字体的图标集，可以通过 Icon 组件使用。

## 引入

```typescript
import { WuIcon } from '@itrain/wu-ui'
```

## 所有图标

{% urlembed %}
./wu-ui-font/index.html
{% endurlembed %}

## API

### @Props

| 名称     | 说明     | 类型                  | 是否必需 | 默认值      |
| -------- | -------- | --------------------- | -------- | ----------- |
| name     | 图标名称 | string                | 是       |          |
| fontSize | 字体大小 | ResourceStr &#x7c; number | 否       | 16        |
| color    | 字体颜色 | ResourceColor         | 否       | Color.Black |
